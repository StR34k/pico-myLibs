◆ enableDAC()

int16_t myMCP49x2::enableDAC ( const bool  value,
const bool  resetValue = true,
const bool  firstChannel = CHANNEL_A 
)

Enable / disable the DAC. Uses the shutdown pin to enable / disable the DAC. Returns 0 (NO_ERROR) if set okay, otherwise if the shutdown pin hasn't been defined, an error code is returned. Defaults to enabled. When the shutdown pin is used, both channels are reset to 0. Use resetValue, and firstChannel to set the value back to last value set. If the load functionality is enabled, then load must be called for the channels to be reset.

Parameters
valueTrue = DAC enabled, False = DAC disabled.
resetValueTrue = Set the value to last value set, False = do not set value.
firstChannelFirst channel to reset to last state.
Returns
int16_t Returns 0 (NO_ERROR) if set okay, negative for error code.
