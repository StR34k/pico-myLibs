)

Initialize the chip.

Parameters
configThe config byte to write to the chip, defaults to standby time of 500us, filter off, comms in 4 wire mode.
measCtrlThe measCtrl byte to write to the chip, defaults to temperature =OSR_X1, pressure=OSR_X1, mode = MODE_FORCED.
humCtrlthe humCtrl byte to write to the chip if available, defaults to humidty = OSR_X1
Returns
true if initialized okay.
false if not initialized okay.
