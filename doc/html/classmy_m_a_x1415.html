Initialize the chip. Initializes the SPI bus, and calibrates the ADC channels. Returns 0 (NO_ERROR) if initialized okay, otherwise if an invalid SPI pin was passed during construction an error code is returned. if an invalid reset pin, or data ready pin is passed, hardware reset, and hardware polling is disabled respectivly.

Parameters
initSPITrue = Initialize SPI bus, False = Don't initialize SPI bus.
intClockTrue (default) = Enable internal clock. False = External clock.
clkFreqTrue (default) = Frequency of 2.4576MHz, False = Frequency of 1MHz
updateFreqThe update frequency, one of the FREQ_* values. Defaults to 0x01, (FREQ_25/ FREQ_60HZ) depending on the Clock frequency.
clkDivTrue = Use clock division, False(default) = Do not use clock division. Ignored if using the internal clock.
Returns
int16_t Returns 0 (NO_ERROR) if initialized okay, negative for error code.
