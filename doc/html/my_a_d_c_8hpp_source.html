2  * myADC
3  * Higer level interface to the raspberry pi pico.
4  * // Checks:
5  * validChannel: returns true if channel is valid. (includes temp sensor.)
6  * validPin : returns true if pin is valid.
7  * validUserChannel: returns true if channel is a valid user channel. (doesn't include temp sensor)
8  * // Lookups:
9  * pinToChannel: convert a pin number to a channel number: returns positive channel number, negative error code.
10  * channelToPin: convert a channel number to a pin number: returns positive: pin number, negative error code.
11  * // Initialize:
12  * initialize : calles adc_init()
13  * initPin : calls adc_gpio_init() for the pin. returns 0 for okay, negative for error code.
14  * initChannel : calls adc_gpio_init() for the associated pin for the channel, returns 0 for okay, negative for error code.
15  * initTemperature: Initialize the temperature sensor.
16  * deinitTemperature: de initialize the temperature sensor.
17  * // Read Raw:
18  * readChannelRaw: return the raw value for the channel. returns positvie for value, negative for error code.
19  * readPinRaw: return the raw value for the channel associated to pin. Returns positive for value, negative for error code.
20  * readTemperatureRaw: return the raw value for the temp sensor. returns positive for raw value.
21  * // Read Converted:
22  * readChannelVoltage: returns the voltage given a channel. Params: float VREF which defautls to 3.3v. return positive for voltage, negaitve for error code.
23  * readPinVoltage: returns the voltage given a pin. Params: float VREF which defaults to 3.3v. return positive for voltage, negative for error code.
24  * readTemperature: returns the temperature. Params: bool returnDegC, true to return degC, false to return degF. returns temperature. or error code. *NOTE: temperatures less than -199.999* collide with error codes.
25  *
26  *
27  */
