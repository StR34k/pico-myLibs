int16_t myAT24C32::write ( const uint16_t  address,
uint8_t *  buffer,
const size_t  length 
)

Write to the EEPROM Preforms a write to the EEPROM. Returns 0 (NO_ERROR) if written okay, otherwise if an invalid address or length are provided, an error code is returned. This blocks until the write is complete.

Note
The EEPROM writes in 32 byte pages only, the data is therefore split up into 32 byte chunks and written. The address counter does not roll over to the first address of the first page on write, so the length of the write can only be to the end of the EEPROM.
Parameters
addressAddress to start write at.
bufferData buffer to write.
lengthNumber of bytes to write. Valid values are determined by the space between the address and the end of the EEPROM.
Returns
int16_t Returns 0 (NO_ERROR) if written okay, negative for error code.
