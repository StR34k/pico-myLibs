\begin{DoxyCompactList}\small\item\em A Class to use a 595 shift register. Daisy chained shift registers can be written to by calling write bit / write byte multiple times. If you want to define clear without without defining enable, you can set enable to M\+Y\+\_\+\+N\+O\+T\+\_\+\+A\+\_\+\+P\+IN to disable it.
