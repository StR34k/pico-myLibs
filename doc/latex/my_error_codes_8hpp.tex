Define all error codes used in pico-\/my\+Libs. 

Define all error codes. All the error code defines for the pico-\/my\+Libs. General Error codes are from 0 -\/$>$ -\/99, so they can be cast to any int type. All other error codes must be cast to int16\+\_\+t at minimum.
