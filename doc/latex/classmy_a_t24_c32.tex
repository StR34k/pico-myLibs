Write to the E\+E\+P\+R\+OM Preforms a write to the E\+E\+P\+R\+OM. Returns 0 (N\+O\+\_\+\+E\+R\+R\+OR) if written okay, otherwise if an invalid address or length are provided, an error code is returned. This blocks until the write is complete. 

\begin{DoxyNote}{Note}
The E\+E\+P\+R\+OM writes in 32 byte pages only, the data is therefore split up into 32 byte chunks and written. The address counter does not roll over to the first address of the first page on write, so the length of the write can only be to the end of the E\+E\+P\+R\+OM. 
\end{DoxyNote}
